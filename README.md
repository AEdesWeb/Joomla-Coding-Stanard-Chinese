## 提交翻譯 

1. 請先到 Issue 中發起您想要翻譯的 [檔案] 與 [章節]，避免大家重複工作。
2. Fork 自己的專案，開始翻譯您的部份。
3. 翻譯完成後，請發出 Pull-Request 並用井號 (#) 連結原有的 Issue。
